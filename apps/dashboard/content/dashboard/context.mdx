# Context Search

Search through your knowledge base to find relevant context and information using our powerful search interface. This page allows you to:

- **Search Content**: Enter queries to find relevant documents and information
- **Visual Results**: View search results in an organized grid layout
- **Copy Results**: Export search results as JSON for further analysis
- **Interactive Cards**: Each result shows title, type, summary, and metadata tags

The search functionality integrates with your entire knowledge base, providing fast and accurate results with relevance scoring.
