This guide helps you get started with the Recurse API. We'll focus on the core endpoints you need to add content and search your knowledge base.

---

## Get Your API Credentials

<div className='fd-steps [&_h4]:fd-step'>
#### Create Account
Sign up at [dashboard.recurse.cc](https://dashboard.recurse.cc) if you haven't already.

#### Generate API Key
In your dashboard, go to **API Keys** → **Create New API Key** and create a new key. Give it a descriptive name like "Development Key".

#### Copy Your Key
Save the API key immediately—it's shown only once for security. You'll need this for all API calls.
</div>

---

## Core API Endpoints

| Endpoint | Method | Description |
|----------|--------|-------------|
| [`POST /documents`](/docs/api-documentation/documents/post) | Direct upload | Upload text content directly |
| [`POST /documents/upload`](/docs/api-documentation/documents/upload-post) | URL upload | Upload from URLs or files |
| [`GET /documents`](/docs/api-documentation/documents/get) | List documents | Retrieve your documents with metadata |
| [`GET /search`](/docs/api-documentation/search/get) | Search | Query your knowledge base |

## Upload Content

### Direct Text Upload
Upload text content directly to your knowledge base.

<Tabs items={['Request', 'Response']} defaultIndex={0}>
  <Tab value="Request">
    ```bash
    curl -X POST "https://api.recurse.cc/documents" \
      -H "Content-Type: application/json" \
      -H "X-API-Key: YOUR_API_KEY" \
      -d '{
        "title": "Getting Started Guide",
        "text": "# Getting Started\n\nThis is a simple guide to help you get started with our platform. Here are the basic steps:\n\n1. Sign up for an account\n2. Generate your API key\n3. Make your first API call\n\nThat'\''s it! You'\''re ready to go.",
        "document_type_hint": "guide"
      }'
    ```
  </Tab>

  <Tab value="Response">
    ```json
    {
      "job_id": "doc_ingest_abc123def456",
      "status": "processing",
      "message": "Document uploaded successfully. Processing started.",
      "result": null
    }
    ```
  </Tab>
</Tabs>

### Upload from URL
Upload content from web URLs or files.

<Tabs items={['Request', 'Response']} defaultIndex={0}>
  <Tab value="Request">
    ```bash
    curl -X POST "https://api.recurse.cc/documents/upload" \
      -H "Content-Type: multipart/form-data" \
      -H "X-API-Key: YOUR_API_KEY" \
      -F "url=https://example.com/blog-post.html" \
      -F "document_type_hint=article"
    ```
  </Tab>

  <Tab value="Response">
    ```json
    {
      "job_id": "url_ingest_xyz789abc123",
      "status": "processing",
      "message": "Document upload from URL started successfully.",
      "result": null
    }
    ```
  </Tab>
</Tabs>

## Search Your Knowledge

Query your knowledge base with semantic search and filtering.

<Tabs items={['Request', 'Response']} defaultIndex={0}>
  <Tab value="Request">
