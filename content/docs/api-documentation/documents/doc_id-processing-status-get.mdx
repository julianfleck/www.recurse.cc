Get detailed processing status for a document, including section-level completion.

This endpoint provides insights into document processing completion and identifies
sections that may need retry or maintenance operations.

**Features:**
- **Section-level Status**: Shows which sections completed successfully
- **Processing Gaps**: Identifies missing frames, embeddings, or metadata
- **Resume Capability**: Information needed to resume failed processing
- **Maintenance Ready**: Data for maintenance system operations

**Response Format:**
```json
{
  "document_id": "doc_123",
  "document_exists": true,
  "sections_total": 5,
  "sections_complete": 3,
  "processing_completion_rate": 60.0,
  "sections_needing_processing": [
    {
      "section_id": "section_456",
      "index": 3,
      "title": "Results Section",
      "missing_operations": {
        "frames": false,
        "embeddings": true,
        "metadata": false
      }
    }
  ],
  "recommendations": {
    "action": "resume_processing",
    "priority": "medium",
    "estimated_completion_time": "2-3 minutes"
  }
}
```

**Use Cases:**
- Monitor document processing progress
- Identify and retry failed operations
- Maintenance system automation
- Quality assurance checks
