Add and ingest a document from multiple input sources.

This endpoint accepts documents in various formats:
- **PDF files**: Upload PDF files that will be converted to markdown
- **URLs**: Provide a URL to extract and process web content
- **Text**: Direct text input (plain text or markdown)

## Input Priority
If multiple inputs are provided, they are processed in this order:
1. File upload (PDF)
2. URL
3. Text

## File Upload Example
```bash
curl -X POST http://localhost:8000/v2/documents/upload \
  -F "file=@research_paper.pdf" \
  -F "title=AI Research Paper" \
  -F "document_type_hint=research_paper" \
  -F 'metadata={"author": "Dr. Smith", "year": 2024}'
```

## URL Example
```bash
curl -X POST http://localhost:8000/v2/documents/upload \
  -F "url=https://example.com/article" \
  -F "document_type_hint=article"
```

## Text Example
```bash
curl -X POST http://localhost:8000/v2/documents/upload \
  -F "text=# My Document\n\nThis is the content..." \
  -F "title=My Document"
```
