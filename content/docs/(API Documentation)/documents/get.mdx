List documents with basic pagination and field selection.

**Purpose**: Simple document listing with pagination. For advanced filtering,
sorting, and complex field selection, use the POST /documents/list endpoint.

**Returns**: A paginated list of documents with basic field selection.

## Field Sets

- `basic`: Essential fields only (id, title, summary, type, index)
- `content`: Content fields (id, title, summary, description, type, created_at)
- `metadata`: Metadata fields (id, title, tags, source, author, metadata, type, created_at)
- `relationships`: Graph fields (id, title, relationships, children, type, created_at)
- `all`: All available fields

## Examples

```bash
# Basic document list
GET /documents?page=1&limit=10

# Documents with content fields
GET /documents?page=1&limit=10&field_set=content

# For advanced filtering, use POST /documents/list
```

**Advanced Filtering**: For document filtering by type, tags, authors, date ranges,
or custom field selection, use `POST /documents/list` with the unified filtering system.
