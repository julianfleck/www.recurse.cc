Add a document to vault storage only (no graph processing).

This endpoint stores documents directly to the file system vault without
triggering any semantic processing, frame extraction, or graph operations.
Useful for:
- Bulk document storage
- Raw document backup
- Testing document storage without processing

**Example Request:**
```json
{
  "text": "This is my document content",
  "title": "My Document",
  "metadata": {"author": "John Doe"}
}
```
