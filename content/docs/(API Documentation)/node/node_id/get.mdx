Retrieve a single node with flexible field selection.

**Purpose**: Retrieve a single node with flexible field selection and filtering options.

**Returns**: A single node object with fields controlled by the filters. Used for getting
detailed information about a specific document, section, or frame.

## Field Sets

- `basic`: Essential fields only (id, title, type)
- `content`: Content fields (id, title, text, summary, description, type)
- `metadata`: Metadata fields (id, title, created_at, updated_at, tags, source, type)
- `relationships`: Graph fields (id, title, relationships, parent_ids, child_ids, type)
- `frame_detail`: Frame-specific fields (id, title, text, type, frame_type, slots, confidence_score, summary, children)
- `all`: All available fields

## Custom Field Control

- `include_fields`: Add specific fields beyond the field set
- `exclude_fields`: Remove specific fields from the field set

## Examples

```bash
# Basic node information
GET /node/abc123?field_set=basic

# Content with additional metadata
GET /node/abc123?field_set=content&include_fields=created_at,tags

# All fields except embeddings
GET /node/abc123?field_set=all&exclude_fields=embeddings
```
