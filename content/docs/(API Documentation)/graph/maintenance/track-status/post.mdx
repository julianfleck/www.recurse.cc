Track node processing status for maintenance system.

This endpoint stores processing status information for nodes that failed
or partially completed processing, enabling the maintenance system to
identify and retry failed operations.

**Request Format:**
```json
{
  "node_id": "section_abc123",
  "status": "NEEDS_EMBEDDING_RETRY",
  "error_details": "Rate limit exceeded",
  "retry_count": 1,
  "last_updated": "2024-01-01T12:00:00Z",
  "task_id": "celery_task_456"
}
```

**Status Values:**
- `COMPLETE`: Node fully processed
- `NEEDS_EMBEDDING_RETRY`: Embedding generation failed
- `NEEDS_FRAME_RETRY`: Frame extraction failed
- `NEEDS_METADATA_RETRY`: Metadata extraction failed
- `EMBEDDING_FAILED`: Final embedding failure
- `FRAME_FAILED`: Final frame failure
- `METADATA_FAILED`: Final metadata failure
- `PARTIAL_SUCCESS`: Some operations succeeded
